Программа курсов "Научись Django"

Урок 1.Вводный + основы GIT
Фреймворк Django: что, для чего, как. Python3 и Django 1.9. Основы GIT: init, коммит, цепочка разработки, удаленный репозиторий, проталкивание, вытягивание
Урок 2. Изучаем GIT
Совместное редактирование файла на примере игры, разрешение конфликтов, ветвление, переключение между ветками, мердж, полезные команды
Урок 3. Начало работы с Django
Виртуальные окружения, установка пакета, создание проекта, структура проекта, файл настроек, запуск разработческого сервера, файл требований пакетов.
Урок 4. Промышленные средства разработки
Работа в IDE PyCharm, настройка PyCharm для работы с Django, отладка проекта, интроспекция запросов и исходных кодов. Рефактор кода и ренейминг. Понятие "чистого кода". Ресурсы для получения справок и знаний.
Урок 5. Схема работы Django
Понятие django-приложения (app). Роутер URL. Обьект запроса, обработка представления, обьект ответа. Формирование URL. Практика: создание статической страницы.
Урок 6. Введение в Django ORM.
База данных, таблица БД, модель ОРМ, соответствие полей. Настройка коннекта к БД, создание БД (MySQL или PostrgeSQL), миграции, Django админка
Урок 7. Введение в Class Based Views.
Структура классов, обработка запроса, методы get и post, наследование, TemplateView, обьект контекста и рендеринг шаблона.
Урок 8. Введение в язык шаблонов Django.
Синтаксис, операторы, наследование шаблонов, переопределение блоков. Создание сайта на основе BootStrap. Раздача статических файлов.
Урок 9. Введение в формы
Понятие формы и поля формы, валидация полей, связанная и несвязанная форма, хелпер ModelForm, виджеты полей.
Урок 10-11. Создание своего APP в совместном проекте.
Общий репозиторий, gitflow, набор урл, свои модели и миграции, редактирование данных через админку, отображение данных через фронт.
Урок 12. Тестирование в Django.
Что это и зачем. Тестовые классы, принципы, написание простейших тестов, покрытие тестами своего приложения.
Урок 13. Разворачивание Django проекта на удаленном хостинге
Подготовка файла сеттингов к деплойменту, клонирование проекта, создание и подключение к БД, установка NGNIX, установка UWSGI, создание конфигурационных файлов, настройка логирования, настройка статики и медиа файлов, запуск и останов веб-сервера.
Урок 14. Продвинутый Django ORM.
Связи таблиц, группировка резалт сета, менеджеры модели, ускорение запросов с помощью select и prefetch, сигналы моделей.
Урок 15. Продвинутые Class Based Views.
Миксины, виды классов, хелперы, паттерн проектирования шаблонный метод.
Урок 16. Продвинутый язык шаблонов.
Собственные тэги, инклюды шаблонов, шаблонизатор Jinja2, настройки сеттингов для шаблонов.
Урок 17. Продвинутый Django.
Middleware, регистрация пользователей, система прав.
Урок 18. Django и JavaScript.
Обработка Ajax запросов, генерация JS скриптов.
Итого 18 занятий, 36 академических часов, приблизительно 4 календарных месяца (занятия раз в неделю)

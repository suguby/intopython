Научись программировать на Python и заработай на этом!

Программа курса:

  • вводное, основы git - 1 занятие
  • python (основы, объектная модель, модули и библиотеки, функциональный подход, PEP8) - 3 занятия
  • инструментарий разработчика (PyCharm, virtual enviroment, написание тестов, отладка) - 3 занятия
  • основы django (основы, статич станицы, ОРМ, class based views, jinja2 templates, формы) - 4 занятия
  • делаем проект (распределенная разработка на gitflow, тестирование, деплоймент) - 5 занятий
    • итого 16 занятий, ~4 месяца

Акцент на самообучение - вы читете литературу, смотрите скринкасты по теме, делаете упражнения, коммитите решения, наставник проверяет и корректирует. На общих занятиях наставник отвечает на вопросы и рассказыает непонятные темы. Проект для общей разработки возьмем Open Source, но не очень сложный. Каждому дадим пилить одну фичу (страницу, форму, етс). Фомат обучения такой же как и с упражнениями - вы делаете, коммитите, наставник проверяет и направляет. Обязательное покрытие кода тестами. Потом поменяемся кодом: нужно будет доработать чужую фичу (очень полезный опыт для проф. программиста).

После окончания курса вы:

  • станете уверенно писать программы на python, используюя обьектно-ориентированный подход
  • будете на ты с системой контроля версий GIT
  • научитесь разрабатывать сайты на Django, с начала и до деплоя
  • научитесь использовать проф. инструментарий разработчика
  • получите собственный код на github, который можно будет указывать в резуме
  • сможете поддерживать свой хостинг django-проектов на debian-based линукс сервере
  • будете получать удовольствие от написания устойчивого и красивого кода, покрытого тестами :)

Условия проведения:

  • один раз в неделю по 2 академ часа, т.е. 1.5 часа обычного времени,
  • занятия вечером, после 20:00, по рабочим дням, график определим все вместе
  • занятия проводятся удаленно (google hangouts) скорость канала должна быть не менее 200Кбит
  • операционная система - Linux Mint/Ubuntu/Debian (можно в virtualbox)
  • производительное железо, обучение идет на PyCharm, он требователен к ресурсам (4Гб памяти как минимум + Core i3)
  • обязательное выполнение домашних заданий (под управлением git)

Требования к ученикам:

  • знание любого языка программирования - паскаль, пхп, етс.
  • навыки администрирования ОС Linux Debian - установка пакетов и настройка программ (или изучите самостоятельно в процессе)

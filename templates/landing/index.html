Научись делать сайты на Django и заработай на этом!

Что ты получишь:

  • станешь уверенно писать программы на python, используюя обьектно-ориентированный подход
  • будешь на ты с системой контроля версий GIT
  • научишся разрабатывать сайты на Django, с начала и до деплоя
  • научишся использовать проф. инструментарий разработчика
  • получишь собственный код на github, который можно будет указывать в резуме
  • сможешь поддерживать свой хостинг django-проектов на debian-based линукс сервере
  • будешь получать удовольствие от написания устойчивого и красивого кода, покрытого тестами

Программа курса:

  • вводное, основы git - 2 занятия
  • основы django (основы, статич станицы, ORM, class based views, django templates, формы) - 7 занятий
  • начинаем делать совместный проект (распределенная разработка на gitflow, тестирование, деплоймент) - 4 занятия
  • продвинутый django (ускорение запросов, миксины, регистрация и система прав, обработка Ajax запросов) - 5 занятий
    • итого 18 занятий, ~4 месяца

Как проходят курсы:

  • в течении недели ты смотришь скринкасты по темам,
  • потом ты делаешь домашку, отправляешь на сервер через git, наставник смотрит и комментирует код
  • вопросы наставнику и/или другим студентам ты можешь задавать по slack/skype,
  • один раз в неделю мы все встречаемся виртуально по google hangouts на 2 часа - разбираем непонятки, задаем вопросы,
  • общая встреча проводится после 20:00, по рабочим дням, график определим все вместе
  • Проект для общей разработки возьмем Open Source, но не очень сложный. Каждому дадим пилить одну фичу (страницу, форму, етс).
  • Фомат обучения такой же как и с упражнениями - команда делает, коммитит, наставник проверяет и направляет. Обязательное покрытие кода тестами. Потом поменяемся кодом: нужно будет доработать чужую фичу (очень полезный опыт для проф. программиста).
  • во время разработки общего проекта координация ведется через RedMine

Требования к ученикам и к оборудованию:

  • знание любого языка программирования - паскаль, пхп, етс.
  • навыки администрирования ОС Linux Debian - установка пакетов и настройка программ (или изучите самостоятельно в процессе)
  • скорость канала должна быть не менее 200Кбит
  • операционная система - Linux Mint/Ubuntu/Debian (можно в virtualbox)
  • производительное железо, обучение идет на PyCharm, он требователен к ресурсам (4Гб памяти как минимум + Core i3)
